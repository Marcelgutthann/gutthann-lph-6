# LPH 6 — Checkliste Vorbereitung der Vergabe

**Grundleistung HOAI § 34, Leistungsbild Nr. 6 (Gebäude und Innenräume)**
> "Ermitteln und Zusammenstellen von Mengen als Grundlage für das Aufstellen von Leistungsverzeichnissen unter Verwendung der Beiträge anderer an der Planung fachlich Beteiligter. Aufstellen von Leistungsverzeichnissen mit Leistungsbeschreibungen. Abstimmen und Koordinieren der Leistungsbeschreibungen der an der Planung fachlich Beteiligten."

Status-Codes: ✓ vorhanden · ⚠ unvollständig · ✗ fehlt · — nicht relevant

## 1. Projektstammdaten / Ausschreibungsstrategie

- [ ] LPH 5 Report vorhanden und eingelesen
- [ ] Bauherr entschieden: Einzel- oder Generalvergabe
- [ ] Los-Strategie (Fachlose / Teillose / Gesamtlos)
- [ ] Öffentliche oder private Vergabe festgelegt
- [ ] Beteiligtenliste aktuell

## 2. Vergabeeinheiten / Gewerkeliste

- [ ] Alle Gewerke nach DIN 276 KGR / StLB erfasst
- [ ] Gewerk-Nummern nach Standard (01 Erdarbeiten … 15 Außenanlagen)
- [ ] Fachplaner-Zuständigkeiten je Gewerk geklärt
- [ ] Abgrenzungen zwischen Gewerken (Schnittstellen) definiert

## 3. Leistungsverzeichnis-Struktur

- [ ] LV-Kopfdaten (Projekt, Vergabenummer, Bieter, Submissionstermin)
- [ ] Vorbemerkungen je Gewerk
- [ ] Positionen nach StLB-Bau (standardisierte Leistungsbeschreibung)
- [ ] Mengen mit Einheit
- [ ] Alternativ- und Eventualpositionen gekennzeichnet
- [ ] GAEB-Format (DA 81/82/83) für digitale Übergabe
- [ ] Zuordnung zu DIN 276 KGR

## 4. Massenermittlung

- [ ] Mengen aus Ausführungsplanung LPH 5 extrahiert
- [ ] Bauteilweise / Raumweise / Geschossweise nachvollziehbar
- [ ] Korrekturfaktoren für Verschnitt berücksichtigt
- [ ] Ab-/Aufmaßregeln nach VOB/C beachtet

## 5. Kostenanschlag nach DIN 276

- [ ] KGR 100–700 detailliert
- [ ] Abgleich mit Kostenberechnung LPH 3 (Fortschreibung LPH 4/5)
- [ ] Abweichung ± dokumentiert
- [ ] Risikopuffer / Unvorhergesehenes (ca. 5–10%)
- [ ] Bauherrenseitige Kosten (Grundstück, Planung, Nebenkosten) berücksichtigt

## 6. CPV-Codes (bei öffentlicher Vergabe)

- [ ] EU-Schwelle geprüft (aktuell 5,538 Mio. € für Bauleistungen, Stand 2024–2025)
- [ ] CPV-Code je Gewerk (aus offiziellem Code-Katalog)
- [ ] Bekanntmachung TED / Bund.de vorbereitet

## 7. Vergabeverfahren

- [ ] Verfahrensart (VOB/A § 3): Öffentlich / Beschränkt / Freihändig / Verhandlungsverfahren
- [ ] Eignungskriterien definiert (Referenzen, Mindestumsatz, Personal)
- [ ] Zuschlagskriterien (niedrigster Preis / wirtschaftlich günstigstes Angebot)
- [ ] Nachweise-Liste (Unbedenklichkeit, Versicherung, Präqualifikation)

## 8. Terminplan Ausschreibung

- [ ] Termine je Gewerk: Versand LV, Bieterfragen, Submission, Vergabe
- [ ] Mindestfristen VOB/A eingehalten
- [ ] Bauzeitenplan nach Submission

## 9. Besondere Vertragsbedingungen (BVB) / Zusätzliche (ZVB)

- [ ] Gewährleistungsfrist (4 Jahre VOB / 5 Jahre BGB, projektabhängig)
- [ ] Zahlungsbedingungen / Abschlagszahlungen
- [ ] Vertragsstrafe bei Terminverzug
- [ ] Sicherheiten (Vertragserfüllungs- / Gewährleistungsbürgschaft)
- [ ] Nachunternehmerregelung
- [ ] Mindestlohn-Nachweise
- [ ] Versicherungspflichten

## 10. Bieterlisten je Gewerk

- [ ] Mindestens 3 Bieter je Gewerk (bei beschränkter Vergabe)
- [ ] Präqualifizierte Bieter geprüft
- [ ] Regionale Unternehmen berücksichtigt
- [ ] Frühere Bieter / Referenzen dokumentiert

## 11. Ausschreibungsstand

- [ ] `Ausschreibungsstand.xlsx` gepflegt (analog allen Projekten)
- [ ] Spalten: Gewerk-Nr., Bezeichnung, Status (LV in Arbeit / versandt / Submission / vergeben), LV-Volumen, Vergabesumme
- [ ] Summenzeile DIN 276 KGR 300 + 400

## 12. Offene Punkte

- [ ] Noch fehlende LV-Positionen
- [ ] Ausstehende Fachplaner-Beiträge
- [ ] Bauherrenfreigaben
- [ ] Übergabe an LPH 7

---

## BLOCKIEREND

1. LPH 5 Ausführungsplanung abgeschlossen (zumindest Leitgewerke)
2. Massenermittlung dokumentiert
3. Kostenanschlag DIN 276 erstellt
4. Vergabeverfahren mit AG abgestimmt
5. Bieterlisten vorhanden
